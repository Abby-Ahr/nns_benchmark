==================================SGH Matlab Code==================================

This package contains cleaned up code for the paper: Scalable Graph Hashing with Feature Transformation.

Qing-Yuan Jiang and Wu-Jun Li
Scalable Graph Hashing with Feature Transformation
IJCAI 2015
Author: Qing-Yuan Jiang and Wu-Jun Li
Contact: jiangqy@lamda.nju.edu.cn or qyjiang24@gmail.com


Some important file:
SGH.m: Implements of SGH and computes top-1k precision.
trainSGH.m: Trains Wx by using sequential learning algorithm.

We appreciate any bug report. Thanks!
